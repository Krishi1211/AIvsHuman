# AI Commit Message Detector & History Analyzer

This project trains machine learning and deep learning models to predict whether open-source Git commits were written by humans or generated by AI coding assistants (like ChatGPT and GitHub Copilot). It extracts history from a large array of open-source datasets (like Elasticsearch and Apache Airflow) to trace the historical rise of generative AI usage in real codebases.

## Results & Findings

### 1. Model Evaluation Metrics
We directly compared classical algorithmic approaches using Term Frequency (TF-IDF bag of words) against a fine-tuned deep learning Natural Language Transformer Model (`distilbert-base-uncased`). 

The traditional machine learning models struggled heavily. Because AI assistants write grammatically perfect but emotionally generic variations of the same vocabulary as a human, word-counting algorithms acted slightly worse than random guessing. Only the Transformer pipeline could accurately extract the structural 'AI syntax' from real commits.

| Model                           | Accuracy | F1-Score | Precision | Recall | MSE (Brier) | R² Score |
|:--------------------------------|---------:|---------:|----------:|-------:|------------:|---------:|
| Multinomial Naive Bayes         |   24.44% |   24.86% |    26.16% | 23.68% |      0.3369 |  -0.3518 |
| SVM (Linear)                    |   28.61% |   31.10% |    31.69% | 30.53% |      0.2105 |   0.1554 |
| Logistic Regression             |   31.11% |   32.24% |    33.52% | 31.05% |      0.2936 |  -0.1782 |
| Random Forest                   |   49.44% |   50.81% |    52.22% | 49.47% |      0.2974 |  -0.1933 |
| **DistilBERT (Transformers)**   | **86.39%**|**86.72%**|  **89.39%**|**84.21%**|   **0.1007** | **0.5958** |

### 2. Historical AI AI Usage Over Time
By batching ~3,000 historical project commits dating from 2014 through 2025 across `distilbert`, we traced the history of when code generation became mainstream.
- **Pre-2022**: ~0% Usage.
- **2022**: Usage spikes to **15.75%** exactly correlating with Copilot General Availability and ChatGPT's launch.
- **2023**: AI Boom! Peaks at **25.25%** of evaluated workflow submissions containing heavy AI assistance text logic.
- **2024-2025**: Normalizes consistently down to **13% - 16%** as workflows iterate and stabilize.

## 💻 How to Run the Scripts

### Installation
Ensure that you have installed the core Python requirements:
```bash
pip install torch transformers datasets scikit-learn pandas numpy tqdm matplotlib seaborn tabulate accelerate
```

### 1. Run inference on custom text
You can feed any string into the `train_ai_detector_hf.py` script to get an AI Probability measurement.
```bash
python train_ai_detector_hf.py --predict "Rewrite the primary extraction function to catch timeout exceptions."
```

### 2. Retrain the DistilBERT Model
To re-train the deep learning parameter weights against the datasets stored inside the `results` folder:
```bash
python train_ai_detector_hf.py --train
```

### 3. Generate Evaluation Metric Graphs
You can generate `.png` graphs to visualize and contrast precision vs MSE error variance between Transformers and Random Forest.
```bash
python generate_graphs.py
```

### 4. Run the Cross-Model Competition script
If you wish to test new classical architectures, run the multi-benchmark loop script:
```bash
python train_ai_detector_multi.py
```

### 5. Generate the Historical Open-Source Usage Graphs
If you would like to run the saved Hugging Face model (`saved_hf_detector/`) across chronological chunks of large open source `.csv` databases and map the historical AI boom.
```bash
python plot_ai_usage.py
```

## Directory Structure
- `train_ai_detector_hf.py`: Primary inference and Deep Learning training loop.
- `train_ai_detector_multi.py`: Evaluates naive baselines (SVM, Logistic, Naive Bayes).
- `generate_graphs.py`: Visualizes the model calibration comparisons.
- `plot_ai_usage.py`: Parses big-data datasets against model pipelines over recorded time epochs. 
- `saved_hf_detector/`: The compiled and weighted fine-tuned Hugging Face cache.
